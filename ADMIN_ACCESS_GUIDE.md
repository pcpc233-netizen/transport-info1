# 🔐 관리자 접근 가이드

## ✅ 보안 강화 완료!

모든 관리자 기능이 이제 **비밀번호로 보호**됩니다.

---

## 🔑 관리자 비밀번호

```
bustime2025!admin
```

**⚠️ 이 비밀번호는 절대 공유하지 마세요!**

---

## 📱 관리자 기능 접근 방법

### 1. 자동화 대시보드 (트리플 클릭)
```
1. 웹사이트 접속
2. 상단 파란색 버스 아이콘을 1초 안에 3번 클릭
3. 비밀번호 입력: bustime2025!admin
4. 자동화 대시보드 접속
```

### 2. 관리자 패널 (트리플 클릭)
```
1. 웹사이트 접속
2. 상단 "전국 교통정보 플랫폼" 타이틀을 1초 안에 3번 클릭
3. 비밀번호 입력: bustime2025!admin
4. 관리자 패널 접속
```

**💡 팁**: 마우스를 빠르게 3번 클릭하면 됩니다!

## 🛡️ 보안 기능

### ✅ 구현된 보안
1. **비밀번호 인증** - 모든 관리자 기능
2. **세션 관리** - 1시간 자동 만료

5. **푸터 정리** - 민감한 링크 모두 제거

### ❌ 제거된 것들
- ~~데이터 수집 링크~~ (푸터에서 제거)

- ~~관리자 패널 버튼~~ (푸터에서 제거)

## 📝 사용 팁

### 빠른 접근:
```
1. 즐겨찾기에 bustime.site 저장
2. 접속 후 아이콘 3번 클릭 (자동화)
```

### 비밀번호 관리:
- 안전한 곳에 저장 (1Password, LastPass 등)
- 주기적으로 변경 (3개월마다)
- 절대 공유하지 않기

3. 비밀번호 입력
