# 🔐 관리자 접근 방법 (간단 버전)

## ⚠️ 중요: 현재 상황

볼트에서 변경한 내용은 **자동으로 사이트에 반영되지 않습니다!**

```
1. 볼트에서 코드 수정
2. GitHub에 Push 필요
3. Vercel 자동 배포
4. 실제 사이트에 반영됨
```

---

## 🔒 안전한 관리자 접근 방법

### 방법 1: 비밀 URL 사용 (가장 안전)

```
https://bustime.site/#admin
```

1. 위 URL 접속
2. 비밀번호 입력: `bustime2025!admin`
3. 관리자 대시보드 접속

**장점**:
- ✅ 메인 사이트 코드 건드리지 않음
- ✅ URL만 알면 어디서든 접근 가능
- ✅ 북마크 저장 가능
- ✅ 모바일/데스크톱 모두 사용 가능

**보안**:
- 이 URL을 **절대 공유하지 마세요**
- 브라우저 즐겨찾기에 저장 (비공개)
- 비밀번호 2중 보호

---

## 🚀 배포 방법

### 1. GitHub에 Push
```bash
git add .
git commit -m "Add secure admin page"
git push origin main
```

### 2. Vercel 자동 배포
- Push하면 자동으로 배포됨
- 3-5분 후 사이트에 반영

### 3. 확인
```
https://bustime.site/#admin
```

---

## 📱 사용 방법

### 데스크톱:
```
1. https://bustime.site/#admin 접속
2. 비밀번호: bustime2025!admin
3. 관리자 패널 / 자동화 대시보드 선택
```

### 모바일:
```
1. 모바일 브라우저에서 위 URL 접속
2. 비밀번호 입력
3. 탭으로 전환하며 사용
```

### 즐겨찾기:
```
1. 브라우저 즐겨찾기에 저장
2. 이름: "관리자" (다른 사람이 모르게)
3. 클릭 한 번으로 접근
```

---

## 🛡️ 보안 수준

```
1층: 비밀 URL (#admin) - 일반인 모름
2층: 비밀번호 (bustime2025!admin)
3층: 세션 관리 (1시간 만료)
4층: 시도 제한 (3회 실패 시 5분 잠금)
```

**메인 사이트**:
- ❌ 관리자 링크 없음
- ❌ 버튼 없음
- ❌ 단축키 없음
- ✅ 완전히 깨끗함

---

## ✅ 이 방법이 가장 안전한 이유

1. **메인 사이트 건드리지 않음**
   - 일반 사용자 경험에 영향 없음
   - 버그 위험 없음

2. **완전히 분리됨**
   - 관리자 코드와 일반 코드 분리
   - 해시 라우팅 (#admin)으로 접근

3. **사용하기 쉬움**
   - URL 북마크 한 번
   - 클릭만으로 접근

4. **안전함**
   - URL 모르면 접근 불가
   - 비밀번호 2중 보호
   - 세션 자동 만료
