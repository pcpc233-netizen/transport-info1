## 핵심 원칙

### 1. 가치 우선
```
"먼저 사랑받는 제품을 만들어라.
 수익화는 나중에 생각해도 늦지 않다."

- Notion: 3년 무료 → 유니콘
- Slack: 무료 → $27B 매각
- Dropbox: 무료 → IPO
```

### 2. 10x 더 좋아야
```
"경쟁자보다 10% 좋으면 안 된다.
 10배 좋아야 사용자가 바꾼다."

네이버 vs 우리:
- 네이버: 광고 많음, 느림, 복잡함
- 우리: 광고 없음(초기), 빠름, 깔끔함
```

### 3. 커뮤니티 > 고객
```
초기에는 "고객"이 아니라 "커뮤니티"
- 피드백 적극 수용
- 사용자 이름 기억
- 개인화된 응답
- 함께 만드는 느낌
```
